## Example projects:

- [fibs-hello-world](https://github.com/floooh/fibs-hello-world) - the vanilla hello-world project plus WASI support
- [fibs-sokol-test](https://github.com/floooh/fibs-sokol-test) - a more interesting project with two sokol-based based demos, sokol shader compiler invocation and copying asset files

## Real world projects:

- [doom-sokol](https://github.com/floooh/doom-sokol) - Doom running on top of the sokol headers
- [chips-test](https://github.com/floooh/chips-test) - my 8-bit emulators hobby project
- [v6502r](https://github.com/floooh/v6502r) - 'Visual 6502 Remixed'
- [sokol-tools](https://github.com/floooh/sokol-tools) - the sokol shader compiler

## Utilities, platform support and library collections

- [fibs-utils](https://github.com/floooh/fibs-utils) - opinionated global build options, embedding and copying asset files, sokol shader
compiler integration
- [fibs-platforms](https://github.com/floooh/fibs-platforms) - adds optional target platforms (currently the WASI and Emscripten SDKs)
- [fibs-libs](https://github.com/floooh/fibs-libs) - non-intrusive fibs library wrappers (wip)

## Fibsified libraries

- [dcimgui](https://github.com/floooh/dcimgui) - all-in-one Dear ImGui source distribution (including C bindings and docking branch)
